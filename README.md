# Sistema de Gerenciamento de Incidentes de Segurança

Projeto desenvolvido com o objetivo de analisar, modelar e documentar um sistema voltado ao gerenciamento de incidentes de segurança da informação.

## Visão Geral

O sistema tem como finalidade registrar, acompanhar e gerenciar incidentes de segurança, permitindo maior controle sobre ocorrências, rastreabilidade das ações executadas e apoio ao processo de resposta a incidentes.

## Artefatos Desenvolvidos

* Levantamento e análise de requisitos
* Casos de uso
* Análise de riscos
* Estrutura Analítica do Projeto (EAP)
* Cronograma (Gráfico de Gantt)
* Documentação de concepção e design
* Planejamento de implantação

# UC001 - Reportar Incidente

## Descrição
Este caso de uso descreve como um usuário/cliente reporta um incidente de segurança no sistema.

## Atores
- Usuário/Cliente

## Fluxo Principal

```plantuml
@startuml
actor Usuario

participant "Sistema" as Sistema
participant "Banco de Dados" as BD

Usuario -> Sistema : Informar dados do incidente
activate Sistema

Sistema -> Sistema : Validar informações

Sistema -> BD : Salvar incidente
activate BD
BD --> Sistema : Confirmação de registro
deactivate BD

Sistema --> Usuario : Incidente registrado com sucesso
deactivate Sistema

@enduml

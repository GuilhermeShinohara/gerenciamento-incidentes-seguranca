Aluno: Guilherme Shinohara

Neste exercício construiremos um diagrama de classes utilizando o PlantUML.

## Diagrama

```plantuml
@startuml

class Curso {
  nome
  duracao
}

class Turma {
  codigo
  semestre
}

class Aluno {
  nome
  RA
}

class Professor {
  nome
  titulacao
}

class Avaliacao {
  tipo
  nota
}

Curso "1" --> "0..*" Turma : possui
Turma "1" --> "0..*" Aluno : tem
Professor "1" --> "0..*" Turma : leciona
Turma "1" --> "0..*" Avaliacao : possui

@enduml
